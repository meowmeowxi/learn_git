### 填坑计划第一弹—————Github入门！  
主要记录一些使用技巧，分享一些教程网址。

**1. Git Bush复制、粘贴**  
复制：Ctrl + ins  
粘贴：Shift + ins 

**2. Github新建项目文件夹**   
①名称后添加/,将显示为文件夹    
②文件夹不能为空，添加一个README.md后commit，即可创建文件夹  

**3. Github README.md添加图片**   
本文使用相对路径的方法      
①建立README.md同级文件夹  
②上传图片至文件夹  
③README.md引用 `![vue页面](assets/vue-helloworld.png)`  
其中assets为文件夹名称，vue-helloworld为图片名称
